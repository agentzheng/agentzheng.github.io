Task3

发表于 2018-06-27

Face Detection

Baseline

  1. Install Python3.6 (I recommend install Anaconda rather than the original python, and create a virtual environment of python3.6 )
  2. Configure pip to use Pypi Tsinghua mirror.(global)
  3. Install OpenCV for python.
  4. Open camera, for each frame, detect the face using the OpenCV functions, and draw a rectangle around the detected face.
  5. Figure out the principle behind the code example, and search for information about the latest face detection methods.(submit in the format of word)

Advanced

  1. Evaluate the performance of your face detection program:
    • How about the detection precision? How to improve it?
    • How much time will be cost to detect a face in an image, How to improve it?
  2. Try to use OpenCV to detect objects other than human face.

Reference

  • https://google.com

  • https://baidu.com

  • https://github.com
